********************
Intégration continue
********************

Jenkins
*******

.. note:: d'après https://wiki.jenkins-ci.org/display/JENKINS/Meet+Jenkins

Jenkins est un outil open source d'intégration continue. Jenkins permet de construire et tester une application de manière continue, laissant aux développeurs la possibilité d'intégrer les changements plus facilement et aux utilisateurs de récupérer un build récent sans problème.

Fonctionnalités
===============

Ci-dessous les fonctionnalités majeures de Jenkins, il est possible d'en ajouter de nombreuses autres à travers des plugins : 

# Facile et rapide à installer
# Facile à configurer grâce à son interface graphique simple et intuitive
# Un écosystème de plugins très complet
# Facile à étendre, la création de nouveaux plugins est simple
# Construction d'application distribuée sur différents OS
