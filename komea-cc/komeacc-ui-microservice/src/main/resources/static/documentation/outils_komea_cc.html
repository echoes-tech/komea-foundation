Komea-CC¶

Introduction¶

KOMEA-CC (Commande et Controle) a pour mission de présenter une interface unifiée vers les outils de l’usine. Les principales actions sont proposées et exécutées avec pour objectifs de

  • simplifier l’usage d’outils complexes
  • maintenir la cohérence des projets entre tous les outils

C’est KOMEA-CC qui se charge de faire executer les actions par les outils et de présenter de manière synthétique les résultats des actions. L’interface permet de visualiser le statut d’un projet d’un seul coup d’oeil.

Exigences en terme d’ergonomie¶

Certaines actions peuvent être relativement longues (créer un hôte virtuel, lancer des tests automatiques, lancer un build, ...). Il faudra être en mesure d’executer les actions en taches de fond et d’afficher leur état d’avancement.

En cas d’erreur dans un outil, les messages d’erreur doivent être affichés de manière complète. Les résultats des actions doivent tous être écrits dans un journal.

Exigences en terme de gestion des droits¶

  • Les projets sont accessibles par tout utilisateur avec tous les droits.
  • Un annuaire LDAP est embarqué avec le serveur et assure l’authentification dans l’usine.

Cas : arrivée sur la page index¶

  • non loggé : on affiche la page de login KSF. On authentifie l’utilisateur dans le LDAP.
  • loggé : on affiche la liste des projets (s’il y a un seul projet, on affiche la page d’accueil du projet)

Cas : création d’un projet¶

l’utilisateur appuie sur le bouton (+) dans la barre de navigation on affiche le formulaire de création d’un nouveau projet avec les boutons apportés par les plug-ins.

L’utilisateur clique sur Create Project :

Après saisie des infos générales, chaque plug-in expose (si nécessaire) un formulaire avec ses informations propres.

Cas : on affiche la page d’accueil du projet¶

La page d’accueil contient :
  • la barre de navigation
  • les propriétés du projet + la liste des actions possibles (= actions impémentées par au moins un plug-in)
  • pour chaque plug-in, un “widget” si celui-ci en expose un.
